Pre-compiling your CSS?

CSS preprocessors are programs that generate CSS out of an own syntax.

I used for the most, nesting. The css code that I needed to write was considerable smaller that what it would have been if I used regular CSS.

Some of the benefits of using a CSS pre-processor:

  • Less CSS to maintain.
  • DRY CSS
  • Better organized CSS
  • You can add some logic and use variables and make calculations for generating CSS.
  • According to this article: https://www.quora.com/What-are-the-advantages-disadvantages-of-using-CSS-pre-processors , it solves browser compatibility problems.

Some disadvantages:

  • You need to learn a new way to write the css, if you want to take full advantage of its features.
  • Debbuging is harder.
  • Increases building time.
